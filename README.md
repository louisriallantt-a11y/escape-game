# 🕵️ Escape Game : Le Manoir Goorse

Bienvenue dans le dépôt de l'Escape Game interactif conçu pour deux équipes (**Lumière** et **Ombre**).

## 🚀 Comment lancer le jeu
1. **Installation locale** :
   - Installez Python.
   - Lancez `pip install -r requirements.txt`.
   - Exécutez `streamlit run app.py`.

2. **Déploiement en ligne** :
   - Connectez ce dépôt à [Streamlit Cloud](https://streamlit.io/cloud).
